Teste Google Analytics

Esta é uma página de teste para verificar se o Google Analytics está funcionando.

Instruções de Teste:

  1. Abra o Console do navegador (F12)
  2. Verifique se aparece "✅ Google Analytics carregado com sucesso!"
  3. Clique no botão "Testar Evento"
  4. Vá para Google Analytics → Relatórios → Tempo Real
  5. Você deve ver esta visita aparecendo
