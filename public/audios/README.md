# Pasta de Áudios

Coloque seus arquivos de áudio de hipnoterapia nesta pasta.

## Formato de Nomenclatura

Os arquivos devem seguir este padrão:
- `sessao-1.mp3` - Introdução à Hipnoterapia para Emagrecimento
- `sessao-2.mp3` - Reprogramando Sua Relação com a Comida
- `sessao-3.mp3` - Liberando Bloqueios Emocionais
- `sessao-4.mp3` - Aumentando Sua Motivação e Disciplina
- `sessao-5.mp3` - Controle da Ansiedade e Estresse
- `sessao-6.mp3` - Visualização do Seu Corpo Ideal
- `sessao-7.mp3` - Sono Profundo e Regenerador
- `sessao-8.mp3` - Autoestima e Amor Próprio

## Formatos Suportados

- MP3 (recomendado)
- WAV
- M4A
- OGG

## Como Adicionar os Áudios

1. Copie seus arquivos de áudio para esta pasta
2. Renomeie-os seguindo o padrão acima (sessao-1.mp3, sessao-2.mp3, etc.)
3. Os áudios aparecerão automaticamente na área de membros

## Exemplo de Estrutura

```
public/
  └── audios/
      ├── sessao-1.mp3
      ├── sessao-2.mp3
      ├── sessao-3.mp3
      ├── sessao-4.mp3
      ├── sessao-5.mp3
      ├── sessao-6.mp3
      ├── sessao-7.mp3
      └── sessao-8.mp3
```

## Onde Encontrar os Áudios?

Se você ainda não tem os áudios, você pode:
- Gravar suas próprias sessões de hipnoterapia
- Contratar um profissional de hipnoterapia
- Usar serviços de Text-to-Speech (TTS) para criar versões de teste
